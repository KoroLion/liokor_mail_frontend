### HowTo Run:
0. npm install -g yarn
1. yarn
2. yarn dev
3. Теперь сайт доступен по localhost:8080

### HowTo Lint:
1. yarn lint
2. yarn lint --fix

### HowTo Deploy:
1. yarn dist
